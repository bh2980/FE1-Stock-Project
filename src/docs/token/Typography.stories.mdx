## **접근성**

W3C의 웹 접근성 지침인 WCAG 2.1에서는 Line Height를 글씨 크기의 최소 1.5배 이상으로 규정하고 있습니다.  
이에 맞춰 치스톡의 Typography 역시 Line Height가 Font Size의 1.5배 이상으로 설정되었습니다.  
(예외 : s 사이즈 1.42배)

글씨가 너무 작아 읽지 못하는 상황을 피하기 위해 xs 사이즈의 사용을 지양하기 바랍니다.
