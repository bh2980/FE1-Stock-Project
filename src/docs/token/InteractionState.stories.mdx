# Interaction State

사용자와 상호작용 시 상태를 정의합니다.  
Overlay 색상은 기본적으로 컴포넌트의 text color를 사용합니다.

<br />

## Enabled

Enabled 상태는 컴포넌트의 기본 상태를 나타내며, 컴포넌트가 켜져있고, 동작할 준비가 되었음을 나타냅니다.

<Canvas withSource="none">
  <StoryWrapper>
    <Button>Enabled</Button>
    <TextField label="Label" />
  </StoryWrapper>
</Canvas>

## Disabled

Disabled 상태는 유저의 상호작용을 금지하는 상태입니다.  
컴포넌트는 꺼져있고, 아무런 응답도 하지 않습니다.  
커서는 금지 표시로 변경되어 상호작용 할 수 없음을 알립니다.

컴포넌트의 배경색을 투명으로 변경하고, 상호작용을 표시하기 위한 div Overlay를 컴포넌트 위에 덮어씌워 구현합니다.  
Overlay의 배경색은 surface-on 색상으로 투명도는 20%로 설정됩니다.  
내부 텍스트 컬러는 surface-on 색상으로 40%의 투명도로 설정됩니다.  
Border는 outline 색상으로 투명도 20%로 설정됩니다.

<Canvas withSource="none">
  <StoryWrapper>
    <Button disabled>Disabled</Button>
    <TextField label="Label" disabled />
  </StoryWrapper>
</Canvas>

## Hover

Hover 상태는 사용자가 마우스를 컴포넌트에 올렸음을 시각적으로 알리기 위한 상태입니다.  
커서의 상태는 포인터로 변경됩니다.

상호작용을 표시하기 위한 div Overlay를 컴포넌트 위에 덮어씌워 구현합니다.  
Overlay의 배경색은 컴포넌트의 텍스트 색과 동일하며, 투명도는 20%로 설정됩니다.

## Press

Press 상태는 사용자가 컴포넌트를 클릭 혹은 터치했음을 시각적으로 알리기 위한 상태입니다.  
간혈적인 상태로 눌렀을 때 잠깐만 표시됩니다.

상호작용을 표시하기 위한 div Overlay를 컴포넌트 위에 덮어씌워 구현합니다.  
Overlay의 배경색은 컴포넌트의 텍스트 색과 동일하며, 투명도는 20%로 설정됩니다.

## Focus & Focus Visible

Focus 상태는 화면 상의 어떤 항목에 포커스가 있는지 명확하게 보여주기 위한 상태입니다.

Focus ring은 컴포넌트 자체에서 구현하며, primary 색상을 사용합니다.

Overlay는 div Overlay를 컴포넌트 위에 덮어씌워 구현합니다.  
Overlay의 배경색은 컴포넌트의 텍스트 색과 동일하며, 투명도는 10%로 설정됩니다.

focusOutlineOffset 속성을 통해 Focus Ring과 컴포넌트 사이 간격 여부를 설정할 수 있습니다.
