## **접근성**

W3C의 웹 접근성 지침인 WCAG 2.1에 따라 명암비를 준수하도록 설정되었습니다.  
베이스 색과 콘텐츠 색(-on)은 최소 4.5의 명암비를 갖도록 설정되었으며,  
primary, secondary, error 색상의 경우 surface-container-highest 색상으로부터 4.5의 명암비를 갖도록 설정되었습니다.  
outline 색상의 경우 surface-container-highest로부터 3의 명암비를 갖습니다.
